Colores CSS

 

 

 

 

 

 

    Podemos referenciar el color mediante su nombre, un código hexadecimal, o código RGB, como se explica a continuación:

  • Color ej: red, blue, green...
  • Código hexadecimal: #rrggbb o #rgb

    Valores válidos son del 0-9 y A-F

    Si los valores se repiten, se puede abreviar #22aaff = #2af

  • Código rgb +a (opcional)

    Valores válidos para rgb: 0...255

    Valores válidos para a (alpha o canal de transparencia):

    0...1, siendo 0 totalmente transparente y 1 totalmente sólido.

    rgb(255,77,25)

    rgb(255,77,25, 0.75)
