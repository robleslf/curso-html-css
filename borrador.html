Ejercicio con selectores de atributos

Página web con un div, con cinco párrafos, y cada uo tenga un atributo que sea mi_bool con valores True o False alternándose.

True

False

True

False

True

---HTML 
    <div>
      <p my_bool="True">True</p>
      <p my_bool="False">False</p>
      <p my_bool="True">True</p>
      <p my_bool="False">False</p>
      <p my_bool="True">True</p>
---CSS 
    img {
      width: 300px;
      height: auto;
    }
    
    [my_bool="True"] {
      color: green;
    }
    
    [my_bool="False"] {
      color: crimson;
    }
