<label>

El Elemento HTML <label> representa una etiqueta para un elemento en una interfaz de usuario.

Este puede estar asociado con un control ya sea mediante la utilizacion del atributo for, o ubicando el control dentro del elemento label. Tal control es llamado "el control etiquetado" del elemento label.

  • for

    El ID del elemento de formulario etiquetable relacionado en el mismo documento que el elemento label.

    En nuestro caso observa que el id del primer input es "nombre", y ese mismo valor es el que se va a utilizar para el atributo for en label

    Podríamos decir: asigna una etiqueta para el id=XYZ

    El primer elemento en el documento con tal ID que coincida con el dispuesto en el atributo for será el control etiquetado para este elemento.

<input>

Atributos <input>

  • id

    Identificador único

  • type

    Tipo de control a mostrar, en este caso un radio button (para elegir una sola opción)

  • name

    Nombre para agrupar los distintos botones del formulario que pertenecen a un mismo grupo, categoría, conjunto

    Observa en nuestro ejemplo que los tres botones poseen este atributo idénticamente, reflejando así que cualquiera de estos inputs pertenecen a una misma categoría

  • value="ABC"

    Valor que recibirá el servidor cuando se seleccione el input

    En este ejemplo el servidor recibirá el string ABC si este input ha sido seleccionado por el usuario
