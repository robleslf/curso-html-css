Selectores de id y de atributo

Selector de ID

Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.

Sintaxis: #idname

Ejemplo: #toc se aplicará a cualquier elemento que tenga el ID "toc".

Este es mi elemento

---CSS

        #mi-elemento {
          /* Estilos CSS para el elemento con ID "mi-elemento" */
          background-color: #ff0000;
          color: #ffffff;
        }

---HTML

      <div id="mi-elemento">Este es mi elemento</div>

Selector de atributo

Selecciona elementos basándose en el valor de un determinado atributo.

Sintaxis: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]

Ejemplo: [autoplay] seleccionará todos los elementos que tengan el atributo "autoplay" establecido (a cualquier valor).

Este es un párrafo importante.

Este es un párrafo normal.

---CSS

        p[data-type="important"] {
          font-weight: bold;
          color: red;
        }

---HTML

        <p data-type="important">Este es un párrafo importante.</p>
        <p>Este es un párrafo normal.</p>

En este ejemplo, todos los elementos <p> que tengan el atributo data-type con el valor "important" se verán afectados por las reglas CSS definidas en el selector [p][data-type="important"], que incluyen un aumento de peso en la fuente a "bold" y un cambio de color a "red". Los párrafos sin este atributo o con un valor diferente no se verán afectados por estas reglas CSS.
